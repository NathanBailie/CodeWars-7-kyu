# 📝 Integers Recreation One

## 🔗 [Task description on CodeWars](https://www.codewars.com/kata/55aa075506463dac6600010d)

### 💡 Solution

```javascript
function listSquared(m, n) {
    let res = [];
    for (let i = m; i < n; i++) {
        let sum = 0;
        for (let j = 1; j <= i; j++) if (i % j === 0) sum += j * j;
        if (Math.sqrt(sum) % 1 === 0) res.push([i, sum])
    }
    return res;
}
```
